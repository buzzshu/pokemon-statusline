The Pokemon Company. Sprites are derived from official Game Freak pixel art.

Files are gen-1 (id 1..151) only; the upstream repo carries gen 1-8.
Filename = `{dex-id}.txt` (e.g., `25.txt` = Pikachu). Each file contains
ANSI 24-bit color escape sequences using U+2580 (upper half block) for
2-vertical-pixels-per-character rendering. Display in any terminal that
supports truecolor + UTF-8 (Windows Terminal, iTerm2, modern xterm).
