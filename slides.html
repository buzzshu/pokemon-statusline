一行裝完

git clone https://github.com/buzzshu/pokemon-statusline.git
cd pokemon-statusline
powershell -ExecutionPolicy Bypass -File install.ps1
  • 自動 patch ~/.claude/settings.json（加 statusLine + permissions.allow）
  • 複製引擎、151 sprites、Pokédex 資料到 ~/.claude/
  • 保留現有 gacha-state.json — 重灌不會洗你的進度
  • 開新 Claude Code session 就 live

需要 Claude Code 2.1+ · Windows Terminal · PowerShell 5.1+

三大主軸

🎴 抽卡收集

151 隻 Gen 1。 Rarity C 90% / U 7% / R 3%；HR 神獸只在 wild encounter 出現。1/4096 shiny。

每天輪換主題屬性，30% 機率重抽到該屬性。

⚔️ 養成戰鬥

每隻 slot 獨立 EXP。 Buddy 每 session +1，其他成員打 gym 才會練。三角等級曲線 L=⌊√(2·exp)⌋+1。

canonical Gen 1 進化等級（妙蛙 L16、暴鯉 L20、超夢不進化）。

🏆 道館闖關

8 個 Gen 1 道館。 小剛 L14 → 阪木 L50，每個給對應徽章 glyph (◆◇✦❀★♥▲●)。

真 type chart 戰鬥引擎、speed priority、1/16 crit。

戰鬥引擎

  • 15 屬性 type chart（Gen 1 canonical：含 Ghost 對 Psychic 0× 的歷史 bug 修正版）
  • 18 隻 flagship signature 招式（御三家終形、5 神獸、皮卡丘、卡比獸、超夢、夢幻⋯）
  • 其他 133 隻自動衍生招式（type1 弱招 + stage≥1 強招 + 副屬性弱招）
  • AI 每回合算 power × typeMul 挑最佳招 — 雙屬性自動切到剋對手的那邊
  • STAB 1.5×、1/16 crit ×2、speed priority、0.85–1.0 隨機係數

開始玩

git clone https://github.com/buzzshu/pokemon-statusline.git
cd pokemon-statusline
powershell -ExecutionPolicy Bypass -File install.ps1

開新 Claude Code session，狀態列立刻變身寶可夢卡帶

/gacha pull  → 抽你的第一張卡
/gacha buddy <id>  → 設為 buddy 開始累 exp
/gacha gym 1  → 衝灰色道館拿第一顆徽章

每跑 $1 USD 的 Claude Code session 就自動 +1 coin —
